## 使用说明

1. 根据任务场景选择上表对应场景
2. 从 `design-styles.md` 选择匹配风格
3. 组合 `[风格DNA] + [场景模板] + [具体内容]`
4. 执行：HTML 生成 / AI 生图 / 混合

## 快速匹配

```
用户需求
  ├─ 做电商 → 上面「电商类」
  │   ├─ 详情页、Banner、主图 → E1/E3 模板
  │   └─ 直播、种草 → E2 模板 + 23 社交美学
  │
  ├─ 做品牌 → 上面「企业品牌类」
  │   ├─ 发布会 → 25 中式发布
  │   ├─ PPT → 22 大厂风 / 10 Swiss
  │   └─ VI展示 → 21 国潮 / 18 Kenya Hara
  │
  ├─ 做内容 → 上面「内容营销类」
  │   ├─ 公众号 → 17 Takram 或 18 Kenya Hara
  │   └─ 小红书 → 23 社交美学
  │
  └─ 做产品 → 上面「数字产品类」
      └─ UI → 22 大厂风 + 原型组件
```
