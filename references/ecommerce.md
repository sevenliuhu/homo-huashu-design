## E1. 天猫/京东详情页

**规格**：
- 宽度 750px（天猫无线）/ 990px（京东PC，桌面为主）
- 高度不限，长页面滚动设计
- 首屏高度 ≈ 屏幕高度（决定跳失率的关键）

**页面结构**（按出现顺序）：

```
┌──────────────────────┐
│  E1-a 首屏主图区       │ ← 最强视觉冲击 / 白底图+产品名
├──────────────────────┤
│  E1-b 核心卖点矩阵     │ ← 3-6 个图标+短文案卖点
├──────────────────────┤
│  E1-c 产品参数对比     │ ← 表格 vs 竞品/vs 上代
├──────────────────────┤
│  E1-d 细节特写         │ ← 材质/工艺/功能放大
├──────────────────────┤
│  E1-e 使用场景展示     │ ← 3-5 个真实场景（用户痛点）
├──────────────────────┤
│  E1-f 用户口碑墙       │ ← 好评截图/买家秀/权威认证
├──────────────────────┤
│  E1-g 品牌故事         │ ← 为什么值得信任
├──────────────────────┤
│  E1-h 规格选择+购买区   │ ← 永远可以跳转购买
└──────────────────────┘
```

**设计要点**：
- 移动端 750px 宽度设计，桌面等比放大
- 首屏3秒决定用户是否继续往下看
- 卖点图标使用真实品牌色，不凭空造色
- 对比表格注意中文对齐（数字右对齐，文字左对齐）
- 留白可以比西方设计少（中国用户习惯信息密度高）
- 信任信号（销量/评分/资质）自然嵌入，不突兀
- 全程可使用 CSS Scroll Snap 实现丝滑滚动

**推荐风格**：
- 国货美妆/食品 → 21 东方国潮
- 科技/3C/家电 → 22 大厂风 或 25 中式发布
- 国际品牌 → 01 Pentagram / 11 Build
- 性价比/拼多多 → 24 中式电商

**代码结构参考**：

## E2. 抖音/快手直播预告

**规格**：
- 竖版 1080×1920px（抖音标准 9:16）
- 每张图只传达1个信息点
- 文字在3秒内可读完

**布局模板**：

```
┌─ 顶部品牌字 ──────────┐
│  品牌 Logo（左上角）   │
├───────────────────────┤
│                       │
│   产品主图（居中）      │
│   【大视觉占60%】      │
│                       │
├───────────────────────┤
│  今晚8点              │ ← 时间（大字号 48px+）
│  ⭐ 新品首发 ⭐        │ ← 爆点标签
├───────────────────────┤
│  限量100份 · 领券减50  │ ← 紧迫感
├───────────────────────┤
│  [点击预约直播]        │ ← CTA 按钮
│  左下：主播头像+昵称  │
└───────────────────────┘
```

**设计要点**：
- 一图一个卖点（不要试图讲所有卖点）
- 文字极度精简（标题最多 15 字）
- 产品图要清晰可辨（点击率核心）
- 时间信息放在视觉中心（而非角落）
- CTA 按钮要足够大（拇指可点）
- 主播头像增强信任感

**推荐风格**：23 社交平台美学 / 24 中式电商

## E3. 电商大促 Banner

**规格**：
- 750×420px（天猫主Banner）
- 320×320px（竖版方形）
- 1280×720px（开屏/首页全屏）
- 文字不超过 15 字（移动端影响点击率的重要因素）

**布局模板**：

```
   满199减30 / 第二件半价      ← 促销力度（最大字号）
   ★ 618年中大促 ★             ← 大促IP
   
   [产品主图（右置）] ←→ [CTA 按钮]
   限量前500名赠礼品             ← 紧迫感
   品牌 Logo（角落水印）
```

**设计要点**：
- 3秒传递完整信息
- 促销力度 > 品牌 Logo（但 Logo 必须可见）
- 紧迫感元素（限时/限量）用对比色
- 产品图占 40-50% 视觉
- 背景保持简洁（复杂背景降点击率）
- 移动端缩略时可辨识

**推荐风格**：
- 618/双11 大促 → 24 中式电商（红橙主调）
- 品牌周年/新品 → 21 东方国潮
- 国际品牌大促 → 01 Pentagram / 11 Build

## E4. 电商品牌海报（小红书店/朋友圈）

**规格**：
- 1080×1440px（3:4 小红书）
- 1200×628px（微信分享缩略图）
- 1080×1080px（朋友圈）

**设计要点**：
- 品牌调性 > 促销感（与 E3 区别）
- 产品图光影处理（质感优先）
- 品牌 Slogan 明确可见
- 配色严格遵循品牌色板
- 留白比电商详情页更多（体现品质感）
- 避免「满减」「打折」等字眼（用「限时」「专属」替代）

**推荐风格**：21 东方国潮 / 11 Build / 17 Takram / 18 Kenya Hara

## E5. 电商数据分析报告

**规格**：
- PDF / HTML 长图 / 信息图
- 阅读设备以手机为主

**内容模块**：
1. 销售总览（GMV / 订单数 / 客单价）
2. 商品排行（TOP 5 SKU）
3. 流量来源（自然/付费/直播/社群）
4. 用户画像（年龄/性别/城市/偏好）
5. 渠道对比（天猫 vs 抖音 vs 私域）
6. 转化漏斗（曝光→点击→下单→支付）
7. 竞品对标（价格/评分/销量对比）
8. 行动建议

**设计要点**：
- 数据准确（不编造数据）
- 图表类型：柱状图/折线图/饼图/热力图
- 数字格式采用中文习惯（「1.23亿」「876.5万」）
- 配色不要超过 4 种（品牌主色+辅色+强调色+灰色）
- 每个数据点都有明确标注

**推荐风格**：
- 严谨型 → 04 Fathom / 10 Müller-Brockmann
- 易读型 → 22 大厂风 / 17 Takram
- 国货品牌 → 21 东方国潮

## 电商设计专项检查清单

- [ ] 是否声明了 `viewport` 和 `width=device-width`？
- [ ] 750px 设计宽度是否等比缩放到小屏（320px-375px）？
- [ ] 价格信息是否正确使用中文习惯（¥29.90 而非 $29.90）？
- [ ] 原价删除线是否明确？
- [ ] CTA 按钮是否足够大（min 44px touch target）？
- [ ] 字体是否加载了中文 Fallback（sans-serif）？
- [ ] 是否有信任信号（销量/好评率/认证）？
- [ ] 图片是否压缩（考虑国内加载速度）？
- [ ] 微信浏览器兼容（X5 Blink 内核）？
- [ ] 不依赖 Google Fonts（使用国内 CDN 或 base64 字体）？
- [ ] 促销活动的日期/时间是否正确？
- [ ] 品牌 Logo 是否清晰可辨（不拉伸/不模糊）？
