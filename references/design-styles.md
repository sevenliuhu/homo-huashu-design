## 一、信息建筑派（01-04）

### 01. Pentagram - Michael Bierut 风格
**哲学**：字体即语言，网格即思想
**核心特征**：
- 极度克制的颜色（黑白+1品牌色）
- 瑞士网格系统的现代演绎
- 字体排印作为主要视觉语言
- 60%+ 留白
- 适用于：国际品牌进入中国市场时的高端调性

**提示词DNA**：
```
Pentagram/Michael Bierut style:
- Extreme typographic hierarchy
- Swiss grid with precise mathematical spacing
- Black/white + one accent color (#HEX)
- 60%+ whitespace ratio
- Suitable for international brands entering China
```

## 六、中式企业派（21-25）⭐ HOMO 增强

### 21. 东方国潮 · 新中式品牌美学

**哲学**：传统不是包袱，是 IP 金矿
**核心特征**：
- 中国传统色彩体系
  - 主色：胭脂 #C03C3C · 黛蓝 #425066 · 墨黑 #1A1A1A
  - 辅色：月白 #E6EAF0 · 松花 #A5B37A · 竹青 #789262
  - 点缀：泥金 #C5A55A · 银灰 #B0B6C0
- 汉字设计作为核心视觉（字形解构/笔画重组）
- 传统纹样现代化（回纹/云纹/冰裂纹几何化）
- 中式留白（气 = 呼吸的空间）
- 印章式构图（方中带圆，重心偏上）
- 书法线条的抽象化而非直接书写

**提示词DNA**：
```
Chinese trendy / 国潮 aesthetic:
- Traditional Chinese color palette
  - Rouge #C03C3C, Ink blue #425066, Ink black #1A1A1A
  - Moon white #E6EAF0, Pine flower #A5B37A, Bamboo green #789262
- Chinese typography as primary visual element
- Modern interpretation of traditional patterns
- Red + ink black as base, gold/silver accent
- Square format (seal carving inspired)
- Poetic negative space (Qi/气 principle)
- Print-making aesthetic (woodblock feel)
```

**代表品牌**：花西子 / 观夏 / 霸王茶姬 / 茶颜悦色
**最佳电商场景**：天猫旗舰店详情页 / 节日限定包装 / 品牌升级海报
**执行路径**：HTML 排版 + AI 生成传统纹样

### 22. 互联网大厂风 · 增长黑客 UI

**哲学**：数据驱动的增长，干净但有转化力
**核心特征**：
- 大字号体系：H1 48-96px / H2 32-40px / Body 14-16px
- 毛玻璃效果（backdrop-filter: blur）
- 弥散光效（radial gradient）
- CTA 层级清晰（主/次/文字链）
- 数据仪表盘布局（KPI 卡片排列）
- 品牌专属渐变（非紫蓝万能渐变）
- 微交互动效（hover/点击/加载）

**提示词DNA**：
```
Chinese big-tech UI style:
- Large typography (H1 48-96px, H2 32-40px)
- Glassmorphism (backdrop-filter: blur)
- Diffused glow effects (radial gradients)
- Clear CTA hierarchy
- Dashboard-inspired grids
- Brand-specific gradients only
- Micro-interactions on all interactive elements
- Clean white/light backgrounds with controlled color
```

**代表产品**：微信 / 飞书 / 字节跳动 / 蚂蚁集团
**最佳场景**：SaaS 登录页 / App 推广 / 小程序 / 数据看板
**执行路径**：HTML 代码渲染（需配合真实的组件）

### 23. 社交平台美学 · 小红书/抖音风

**哲学**：信息流中多停留 1 秒，转化率翻倍
**核心特征**：
- 竖版 9:16 优先
- 高对比度 + 柔和光晕（但不过度）
- 封面图决定 60% 的点击率
- 文字 ≤ 15 个中文字
- 生活化 > 精致过度
- warm tone, approachable
- 用户生成内容（UGC）视觉语言
- 评论/点赞/分享等社交元素自然嵌入

**提示词DNA**：
```
Social platform content aesthetic:
- Vertical format (9:16) mobile-first
- High contrast with soft glow
- Cover image as primary attention
- Minimal text (≤ 15 Chinese characters)
- Lifestyle feel, not overly polished
- Warm tone, approachable, genuine
- UGC visual language
- Native feed integration design
```

**代表平台**：小红书 / 抖音 / B站
**最佳电商场景**：种草图文 / 短视频封面 / 直播预告
**执行路径**：AI 生成为主（Midjourney / DALL-E），HTML 辅助排版

### 24. 中式电商直击 · 转化率驱动

**哲学**：每一帧都在卖，但不硬卖
**核心特征**：
- 价格信息视觉武器化
  - 原价划掉（text-decoration: line-through）
  - 到手价 放大 300%
  - 优惠差用对比色（红字/橙字）
- 信任信号嵌入：销量 ≥ 10万 / 好评率 99% / 官方认证
- 场景化展示：使用中 > 白底产品图
- 紧迫感设计：倒计时 / 限量进度条 / 仅剩XX件
- 信息密度高：中国人习惯看到更多内容（5-8个卖点）
- 移动端竖版优先

**提示词DNA**：
```
Chinese e-commerce conversion design:
- Price as hero element (strikethrough original, 300% sale price)
- Trust signals (sales 10万+, rating 99%, verified badges)
- Contextual product display (usage scenarios)
- Urgency elements (countdown, limited stock bars)
- High information density (5-8 selling points)
- Mobile-first vertical layout
- Social proof widgets embedded
- Bundle/套餐 visual representation
- Red/orange for price emphasis, brand color for trust
```

**代表平台**：天猫 / 京东 / 抖音电商 / 拼多多
**最佳场景**：主图 / 详情页 / 促销 Banner / 直播间背景
**执行路径**：HTML 排版（精确控制价格展示）

### 25. 中式科技发布会 · 舞台 Keynote

**哲学**：科技企业的舞台感，每页都是高光时刻
**核心特征**：
- 深色背景（深灰 #1E1E1E / 深蓝 #0A1628 / 黑 #000）
- 超大文本，投影安全字号（40pt+）
- 产品渲染图分层展示
- 数据可视化作为叙事锚点
- 舞台灯光效果（spotlight / glow / radial reveal）
- 中英文混排规范
- 品牌名高光时刻
- 每张页面一个核心信息

**提示词DNA**：
```
Chinese tech keynote design:
- Dark background (charcoal/deep blue/black)
- Oversized typography (40pt+ projection-safe)
- Layered product renderings
- Data visualization as narrative anchors
- Stage lighting effects (spotlight, glow)
- Controlled Chinese-English bilingual layout
- Brand reveal moments
- One core message per slide
- Transition-friendly compositions
- High contrast for projection environments
```

**代表品牌**：华为 / 小米 / OPPO / vivo / 微信公开课
**最佳场景**：新品发布 / 品牌升级 / 年度演讲
**执行路径**：HTML Keynote + AI 生成产品渲染图
