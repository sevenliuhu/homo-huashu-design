# 设计评审深度指南 · 中文企业版

> 5 维度专家评审（原始）+ 2 个附加维度（文化适配度 + 转化力）

---

## 核心 5 维度（原始保留）

### 1. 哲学一致性（Philosophy Alignment）
| 分数 | 标准 |
|------|------|
| 9-10 | 设计完美体现选定哲学内核，每个细节有哲学依据 |
| 7-8 | 方向正确，核心特征到位，个别细节偏离 |
| 5-6 | 能看出意图，但混入其他风格元素 |
| 3-4 | 表面模仿，未理解哲学内核 |
| 1-2 | 与选定哲学基本无关 |

**评审要点**：
- 是否使用了标志性手法？（国潮？大厂风？电商转化？)
- 配色、字体、布局是否符合该语汇体系？
- 有无自相矛盾？（如选国潮却用赛博霓虹）

### 2. 视觉层级（Visual Hierarchy）
**评审判据**：
- 标题与正文字号对比 ≥ 2.5倍（中文建议 3倍）
- 建立 3-4 个清晰层级
- 留白引导视线
- 眯眼测试（层级是否仍然清晰）

### 3. 细节执行（Craft Quality）
**评审判据**：
- 统一间距系统（8pt/4pt网格）
- 颜色 ≤ 3-4 种
- 字体 ≤ 2 种
- 边缘对齐精确

### 4. 功能性（Functionality）
**评审判据**：
- "删掉任意元素会变差吗？"
- CTA/关键信息在最显眼位置
- 无"因为好看所以加上去"的元素

### 5. 创新性（Originality）
**评审判据**：
- 是否避免 cliché？
- 在哲学框架内是否有独特表达？
- 有无「意想不到但合理」的设计决策？

## ⭐ 附加维度 A：文化适配度（中文版特化）

| 分数 | 标准 |
|------|------|
| 9-10 | 完美适配中国文化语境，既有辨识度又不落俗套 |
| 7-8 | 整体方向正确，文化元素使用贴切 |
| 5-6 | 无明显文化错误，但统一性不够 |
| 3-4 | 存在文化元素生搬硬套 |
| 1-2 | 明显不适配（如西式排版直接套中文） |

**评审要点**：
- **字号**：中文 14-16px body / 32px+ 标题（字面率考虑）
- **行高**：中文 1.5-1.8 倍（不同于英文 1.2-1.5）
- **字距**：正文字距 0 / 大标题 -2~2px（字距微调）
- **数字**：万/亿 还是 千分位？
- **日期**：2026年5月22日 vs 05/22/2026
- **文化符号误用**：
  - 红色：庆典/喜庆（√）商业严谨/科技有时拒斥（×）
  - 白色：丧事/死亡（传统）纯洁/现代（当下）
  - 黄色：皇家/尊贵（传统）色情（网络）
- **字体选择**：衬线体 vs 宋体（不是等价的）
- **信息密度**：中国用户接受更高密度（电商尤为明显）

### 文化适配度常见问题

| 问题 | 表现 | 修复 |
|------|------|------|
| 西版中搬 | 英文行高套中文，字距混乱 | 中文行高 1.5-1.8 |
| 假国潮 | 龙/祥云/大红大黄 | 用抽象传统元素替代 |
| 手机号格式 | 写成 +86 10 8888 8888 | 400-888-8888 |
| 地址格式 | Street, City, ZIP | 省市区街道门牌 |
| 价格格式 | $29.99 | ¥29.90 / 29.9元 |
| 品牌名 | 只有英文名 | 中英双名（如 "Apple 苹果"）|

## ⭐ 附加维度 B：转化力（电商专化）

| 分数 | 标准 |
|------|------|
| 9-10 | 每个元素服务于销售转化，零冗余 |
| 7-8 | CTA 清晰，信息传递高效 |
| 5-6 | 基本具备电商要素，缺乏紧迫感 |
| 3-4 | CTA 不明确，信息层级混乱 |
| 1-2 | 完全不适合销售场景 |

**评审要点**：
- 价格信息是否突出？（3秒定位价格位置）
- 优惠信息是否一眼看到？
- CTA 按钮 min 44×44px + 对比色
- 信任信号（销量/评分/认证）是否可见？
- 信息密度是否适合移动端快速消费？
- 紧迫感是否自然？（限时/限量/阶梯价）
- 是否有购买阻力未消除？（运费/退换/售后）

## 中文设计 Top 10 问题（新增专有）

### 1. 字体不对
没有正确的中文 Fallback → 页面全是系统默认字体
**修复**：`font-family: 'PingFang SC', 'Microsoft YaHei', 'Noto Sans SC', sans-serif`

### 2. 英文排版逻辑套中文
行高用 1.2（那是英文的），字距没调整
**修复**：行高 1.5-1.8，字距正文字体使用默认

### 3. 文化符号硬塞
龙/祥云/灯笼 不管什么品牌都放
**修复**：用更抽象的传统元素或品牌自有的 IP 符号

### 4. 颜色过多
试图用颜色区分所有层级，结果视觉混乱
**修复**：限制 1 主色 + 1 辅色 + 1 强调色 + 灰阶

### 5. 信息密度走极端
要么太满（像促销传单）要么太空（像没人要）
**修复**：按场景匹配密度：电商密 / 品牌适中 / 发布会疏

### 6. 尺寸没理解中国显示环境
字号太小（投影看不清）或太大（手机显示不全）
**修复**：手机 14-16px body / 投影 24px+ body

### 7. 价格不突出
用户找了 3 秒才知道多少钱
**修复**：价格放大 3 倍 + 红色/强调色

### 8. 没有信任信号
用户不敢买
**修复**：嵌入销量/好评/认证/晒图

### 9. 英文字母/数字 与中文不协调
英文用 Inter 搭配 思源黑体，字重不匹配
**修复**：中文字体自带的数字/西文优先；或指定同一字重的英文 Fallback

### 10. 双11/618风格错误
"打折"直接砸脸上，没有设计感
**修复**：促销元素融入品牌语汇，而不是跑出两个风格
